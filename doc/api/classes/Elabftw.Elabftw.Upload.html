\Elabftw\ElabftwUpload

What happens when you upload a file. This class has two public methods : uploadFile() for normal upload and uploadLocalFile() for calling it from the ImportZip class.

we receive the file in $file. The array looks like that : name : filename.pdf type : "application/pdf" tmp_name "/tmp/phpLzaurte" error : 0 size 134482
